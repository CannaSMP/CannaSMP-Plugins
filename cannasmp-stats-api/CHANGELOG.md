# Changelog

## 2.0.0

- Rebuilt as modular CannaSMP ServerAPI.
- Added secure versioned REST endpoints.
- Added API key authentication, IP whitelist support, CORS, rate limiting, and structured errors.
- Added server, player, plugin, JVM, CPU, RAM, world, weather, TPS, MSPT, uptime, and integration status snapshots.
- Added graceful reflection-based Vault, LuckPerms, PlaceholderAPI, and EssentialsX compatibility.
- Added GitHub Pages-compatible website snapshot export.
- Added `/serverapi` admin command suite.
- Added public API event/model scaffolding for future plugins.

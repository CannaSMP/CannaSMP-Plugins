# CannaSMP Tools

Utility plugin for CannaSMP.

Features:

- Chat emoji replacement such as `:sob:` and `:skull:`
- Emoji replacement on signs and anvils
- Bad-word filtering in chat and selected text commands
- `/clearchat` and `/cc`
- `/joke`, `/joke submit <joke>`
- `/jokeadd <joke>`, `/jokeremove <id>`, `/jokelist`, `/jokepending`
- Discord webhook notification for submitted jokes

Set `webhook.joke-submissions` in the live config before enabling webhook submissions.

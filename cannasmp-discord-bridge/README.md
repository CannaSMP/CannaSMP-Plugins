# CannaSMP Discord Bridge

Custom Discord bridge for CannaSMP.

Features include:

- Minecraft chat to Discord
- Discord chat to Minecraft
- Staff bridge chat
- Join/leave/death logs
- Minecraft account linking
- LuckPerms rank and Discord role sync
- Discord-side server info commands
- Restricted staff moderation/admin commands

Do not commit live bot tokens or channel IDs. Use placeholders in public configs.
